Cómo te protege CanalSeguro

🚫🆔

Sin datos personales

No pedimos nombre, correo ni teléfono. No hay cuentas ni registro: nada que pueda vincularte con tu denuncia.

🧹

Supresión de metadatos

Las imágenes que adjuntas se reprocesan en tu propio navegador para eliminar datos EXIF como ubicación GPS o modelo del dispositivo.

🔑

Código irrecuperable

Al enviar recibes un código único. Es la única forma de consultar tu caso; ni nosotros podemos regenerarlo.

🔍

Código fuente auditable

Todo el código es público en GitHub para que cualquier persona pueda verificar cómo se protege tu información.

    Acerca de CanalSeguro

    CanalSeguro es un prototipo de canal de denuncia ciudadana cifrada que busca eliminar las barreras de confianza que disuaden el reporte de irregularidades en contextos de alto riesgo.

    Principios

    • Anonimato total e irreversible: ausencia de datos de identificación.
    • Confidencialidad de la evidencia: supresión de metadatos en los archivos.
    • Seguimiento sin identidad: código único e irrecuperable.
    • Transparencia: código fuente público para auditoría independiente.
    • Accesibilidad: interfaz clara, responsiva y con modo de alto contraste.
    Aviso sobre el prototipo. Esta es una demostración funcional desarrollada con fines académicos. El cifrado del lado del servidor y la recepción real de denuncias forman parte del backlog del proyecto; en esta versión las denuncias se guardan únicamente en el almacenamiento local de tu navegador para demostrar el flujo de extremo a extremo.

    Proyecto y metodología

    Desarrollado de forma individual aplicando un enfoque ágil Kanban (con prácticas puntuales de Scrum) para la asignatura Diseño de Aplicaciones Web. El trabajo se gestionó con un tablero Kanban con límites de WIP, una bitácora diaria reflejada en los commits y seguimiento de incidencias mediante GitHub Issues.
